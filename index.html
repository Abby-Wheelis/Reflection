What I've Learned

The hardest thing that I have learned so far is the basics of how to edit html using javascript. Because I have now learned the d3 algorithm, I am able to make additions to the html by using the tools of the d3 library. In order to follow through with a d3 algorithm, there are 6 basic steps.

  1. Select where the new html elements will appear on the page. This is done using the d3 command .select(“CSS selector”). It is possible that you may need to create the space where you want to put the new html elements, such as creating a div to put paragraphs in. This is done using the d3 command .append(“CSS selector”).
  2. Select all of the elements that don’t exist yet, because that is what you are creating. This is done using the d3 command .selectAll(“CSS selector”). You have to be careful here that there are not any of said elements already in existence within the CSS selector that you used in .select. This is where it may be useful to create a new div, either in the previous step or directly in the html. If there was already say, a paragraph, I’ve found that it will “eat” the first paragraph you expected to appear.
  3. Bind the data. This is done using the d3 command .data(name of dataset). It is important that the name of your dataset matches that which you used in your promise and that which you set as a parameter for this function (see the example code below). The data set must be of the array type or the d3 algorithm will not work. However, the array can be made up of any type.
  4. Get all of the missing visual elements. This step will always be the d3 command .enter(). It essentially allows you to append the elements that you are creating using the data set.
  5. Create and append the elements. This step is done using the d3 command .append(CSS selector. It is important that your CSS selector here matches that which you used in .selectAll or you won’t achieve the desired effect.
  6. The final step is to decorate. This is achieved using the d3 commands .text(“string”), .attr(“attribute”, value), and/or .on(“action”, value). These commands often contain functions in the form function(d) {return d.attribute}, for example. It is helpful to remember here that the d parameter represents one of the values in the array, for example, if your array is filled with objects, d represents one of those objects. It is also important to note that you can have many of these decoration commands in your d3 algorithm function.
       var makeExample= function(data)
       {
         d3.select("body")
         .selectAll("div")
         .data(data)
         .enter()
         .append("div")
         .text(function (d)
         { return d.attribute});
       }

This example code would append to the end of the body a div for every element in the data array containing the value of a specified attribute for each element.
